Python function to calculate Binary, Octal or Hexadecimal number from Decimal number
without using Python built-in methods

possible arguments:
    -b  ( for binary number )
    -o  ( for octal number )
    -h  ( for hexadecimal number )


example of usage:

    python main.py -b 100
    python main.py -h 15119
    python main.py -h "15119"
